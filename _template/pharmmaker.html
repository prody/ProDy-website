Pharmmaker for Pharmacophore modeling

Pharmmaker is designed for building pharmacophore model using outputs of druggability simulations (DruGUI). The pharmacophore models can be used for virtual screening of libraries of small molecules. A strong aspect of the method is that Pharmmaker uses multiple target conformations dependent on the binding poses of probes where they interact during druggability simulations, meaning that the binding score in virtual screening can be more evaluated in a more realistic manner. Also, we can have multiple pharmacophore models with different target conformations and probe poses, which can be analyzed statistically.

Features

Pharmmaker can be used to

  • identify high affinity residues on the target protein
  • rank the interaction pairs between residues and probe small molecules
  • select snapshots with the top ranking interaction pairs
  • build a pharmacophore model
  • perform virtual screening as inputs

Download

Pharmmaker files can be obtained from Getting Pharmmaker.

DruGUI files can be obtained from Getting DruGUI.

Reference
Lee JY, Li H, Krieger JM, Bahar I Pharmmaker: Pharmacophore modeling and hit identification based on druggability simulations 2019 Protein Science 29(1):76-86
