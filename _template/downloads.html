Installation

1 Using a Package Manager

You can install ProDy and Evol using pip, pip install -U ProDy. Please note that you may have to manually remove previous versions (<1.10.3) of ProDy first.

2 Requirements
You need to install the following software before installing ProDy:
Python
We recommend that you use 2.7, 3.5 or later, up to 3.11 (or 3.10 for hydrophobic overlaps) For Windows, you can choose 32-bit or 64-bit Python installer. We recommend using Anaconda instead of the stardard Python distribution as it provides the conda package and environment manager as well as access to many useful packages including the interactive Python shell and Jupyter notebook.
NumPy
We recommend that you use the latest version of NumPy, but v1.10 and later are supported.
Biopython
We recommend that you use the latest version of Biopython, but all versions should be supported.
SciPy
We recommend that you use the latest version of SciPy, but all versions should be supported.
3 Installation
  • Linux & Mac OS
  • Windows

If you have pip, you can simply install ProDy as follows:

                    $ pip install prody

This will give you ProDy-2.4.1, which is the latest stable release but does not include InSty.

If you want the latest version of ProDy, you can either clone or manually download and extract the zip file contents from GitHub and run setup.py as follows: 

                    $ git clone https://github.com/prody/ProDy.git
                    $ cd ProDy
                    $ python setup.py build_ext --inplace --force
                    $ pip install -Ue .
                  
                    $ tar -xzf ProDy-main.zip
                    $ cd ProDy-main
                    $ python setup.py build_ext --inplace --force
                    $ pip install -Ue .

Since version 1.10.3, ProDy does not provide a binary installer anymore. Instead, please install ProDy through pip:

or from GitHub as on Linux and Mac. 
                    $ pip install prody
4 Recommendations
We recommend that you also install Matplotlib for plotting, and IPython for interactive usage.
