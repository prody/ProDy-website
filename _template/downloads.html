Installation

* Using a Package Manager

You can install ProDy and Evol using pip, pip install -U ProDy. Please note that you may have to manually remove previous versions of Prody first.

* Requirements
You need to install the following software before installing ProDy:
Python
We recommend that you use 2.7, 3.4 or later. For Windows, you can choose 32-bit or 64-bit Python installer.
NumPy
We recommend that you use the latest version of NumPy, but v1.10 and later are supported. For Windows, make sure that you choose compatible versions of NumPy and ProDy. Make sure that the version is compatible with your ProDy and Python version.
Biopython
We recommend that you use the latest version of biopython, but all versions should be supported.
3 Installation
  • Linux & Mac OS
  • Windows

Extract tarball contents and run setup.py as follows:

$ tar -xzf ProDy-{{ version }}.tar.gz
$ cd ProDy-{{ version }}
$ python setup.py build
$ sudo python setup.py install

To be able use ProDy Applications and Evol Applications in command prompt (cmd.exe), append Python and scripts folders (e.g. C:\Python27 and C:\Python27\Scripts) to PATH environment variable.

4 Recommendations
We recommend that you also install Matplotlib for plotting, and IPython for interactive usage. On Windows, you need to install pyreadline for colorful IPython sessions.
