Dynamics of Protein Complexes from Cryo-EM Maps

CryoDy calculates the dynamics of protein complexes resolved by cryo-EM by approximating the density maps with pseudoatoms.

API Features

You can use CryoDy to

  • parse electron density map files from the EMDataBank
  • apply the topology representing network algorithm to fit pseudoatom beads to cryo-EM maps for use in ENM NMA
  • compare pseudoatom models to other structures and perform PCA to analyse energy landscapes
  • sample conformational transitions using adaptive ANM, which does not not atomic details
  • apply other ENM-based methods such as hitting time analysis
